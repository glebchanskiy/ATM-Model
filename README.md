# ATM Model- LR4

## Install

Clone repository:

```sh
git clone https://github.com/glebchanskiy/PPOIS-sem4-lab4
```

Install python deps:

```sh
cd lab4
poetry install
```

## Run

Run game:

```sh
cd lab4
# run cli atm:
poetry run cli

# run gui atm:
poetry run gui
```

## Dependencies

- poetry
